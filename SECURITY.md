# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Scope

This package provides PHPStan stubs (type definitions only) and does not execute code at runtime. Security risks are minimal, but potential concerns include:

- Malformed PHP syntax causing parser issues
- Incorrect type definitions leading to false sense of type safety

## Reporting a Vulnerability

For security concerns, please email **me@artemsemkin.com** rather than opening a public issue.

You can expect a response within 7 days.
